这边用的是 mihomo,别问,问就是懒得配置 iptables 规则(配置错了还容易炸网络,只能靠 VNC 之类的救回来.)  
不用 WARP 的原因是 f 佬的脚本不再支持 Teams 和 WARP PLUS 账号了,免费版又有 1 GB 的流量限制.加上 Alice 那边也刚好提供了台湾家宽出口索性直接拿来用了.  

### 安装 mihomo

我这边使用的是 Debian 12 系统,理论上 Debian 系列的能通用,也可以参照[虚空终端(被墙)](https://wiki.metacubex.one/)提供的安装教程,或者是自己自行手动安装.  

```bash
# 注意请把 v1.19.19 换成具体的版本号,socks5 代理如果要是 alice 那边有变化的话请自行更改相关信息
curl -x socks5h://alice:alicefofo123..OVO@[2a14:67c0:116::1]:10001 -L https://github.com/MetaCubeX/mihomo/releases/download/v1.19.19/mihomo-linux-amd64-v1.19.19.deb --output mihomo.deb
# 使用 apt 从本地安装 mihomo,如果提示拒绝访问请手动使用 chmod 更改权限
# 更新前请注意备份配置文件
sudo apt install ./mihomo.deb
# 删除安装包
sudo rm -rf mihomo.deb
```

### 编辑 mihomo 配置文件

# 设置访问密码 (建议更改默认密码)

如果你想和 xray 良好共存(防止 xray 流量进 mihomo TUN)的话需要手动创建 xray 账号并更换上述配置文件中排除用户的 UID 为 xray 账号的 UID.本教程不过多叙述.

创建 xray 用户和组.  

```bash
sudo groupadd --system xray
sudo useradd --system --gid xray --no-create-home --shell /sbin/nologin xray
```

手动查看 xray 用户的 UID

```bash
id xray
```

安装/更改 xray 运行的用户.(当然也可以手动编辑 service 文件改.)  

```bash
sudo bash -c "$(curl -L https://github.com/XTLS/Xray-install/raw/main/install-release.sh)" @ install -u xray
```

### 启用 mihomo 服务

```bash
sudo systemctl enable mihomo --now
```

应该,或许你用 `curl ip.sb -4` 就能看到 IPv4 出口的地址了.
