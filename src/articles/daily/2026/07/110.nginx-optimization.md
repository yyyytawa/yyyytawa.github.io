## 调优部分

### 1. 404 页面等不显示 Nginx 版本号

```nginx title="nginx.conf"
    server_tokens off;
```

### 2. 优化 SSL 配置

```nginx title="nginx.conf"
    # SSL 配置
    # 证书文件
    ssl_certificate     /etc/ssl/path/to/fullchain.pem;
    ssl_certificate_key /etc/ssl/path/to/privkey.pem; # 请务必设定用户为 root,权限为 700,防止其他用户读取

    # 加密套件
    ssl_protocols TLSv1.2 TLSv1.3; # TLS 协议版本
    ssl_prefer_server_ciphers on;
    ssl_ciphers 'ECDHE-ECDSA-AES128-GCM-SHA256:ECDHE-RSA-AES128-GCM-SHA256:ECDHE-ECDSA-AES256-GCM-SHA384:ECDHE-RSA-AES256-GCM-SHA384:ECDHE-ECDSA-CHACHA20-POLY1305:ECDHE-RSA-CHACHA20-POLY1305';
    ssl_ecdh_curve X25519MLKEM768:X25519:prime256v1:secp384r1; # 添加支持 PQC 的算法
    # 性能优化配置
    # 启用会话缓存，减少重复握手的开销
    ssl_session_cache shared:SSL:20m;
    ssl_session_timeout 1d;
    ssl_session_tickets on;             # 启用会话票据，进一步提升恢复速度
```

### 3. 默认开启 http2

```nginx title="nginx.conf"
    http2 on;
```

### 4. 拒绝未知 SNI 的 TLS 握手,防止通过 TLS 证书泄露源站 IP

```nginx title="conf.d/default.conf"
server {
    listen  443 ssl default_server;
    listen  [::]:443 ssl default_server;
    server_name  _;

    ssl_reject_handshake on; # 防止证书泄露
}
```
