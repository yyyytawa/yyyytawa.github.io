解决方法(参考 https://github.com/npm/cli/issues/8075 ):  
1. 设置环境变量 COREPACK_INTEGRITY_KEYS=0
2. 更新corepack版本  
```bash
npm install -g corepack@latest
```
3. 重新运行pnpm即可
