# 概述:
1. "我们","我"在此处均指 yyyyt.top 旗下的域名运营的站点.  
2. **对于协议里面的关键内容我们会使用加粗表示.**  
3. 本政策修改后可以不通知,公示时刻起有效.  

# 信息的收集

1. 我们会收集您的 IP 地址,User-Agent(UA),请求 URL 等必须信息,这些信息属于网站提供服务必须内容,若拒绝提供本站无法提供服务.

2. 评论系统基于[giscus](https://giscus.app/),如果您使用评论系统将会收集以下信息  
- IP 地址,UA 等网站基本功能所必须信息
- 您的Github账号的部分信息(如用户名,头像等信息)
- Giscus正常运行的所需其他信息

3. 本网站部署在 Edgeone,Cloudflare 等托管商,相关托管商可能会收集您的隐私信息,建议您阅读相关托管商的隐私政策.

4. 本站可能会在本地存储写入一些内容如 Cloudflare WAF 的留下的信息等,用于保护网站免受攻击.

# 信息的保护

1. **我们会尽力去保护您的个人信息.由于网络的特殊性,我们无法保证 100% 不泄露.但是我们会尽力避免泄露事故的发生,如果因我们方面原因失误造成的泄露由我们负责.对于 Giscus 等第三方或者是本站托管商造成的泄露我们无法负责.**

2. **我们不会向任何第三方透露您的信息**,除非
- 相关法律法规要求将您的信息共享给相关第三方机构
- 回应重大公众事件
